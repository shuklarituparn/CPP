Friend:

A function or a class that has access to the private member of a class
And, that function of our class is not the member of the class that it's accessing

Function:

Can be regular non-member function
Can be member methods of another class

Class:

Another class can have access to private members




**Friendship must be granted not taken
    *Must be explicitely granted
    *Declared explicitely in class granting the friendship
    *Declared in the function prototype with the keyword FRIEND



*Friendship is not symmetric
    *If A is a friend of B
    *Doesn't mean that B is a friend of A

*Friendship is not transitive
    *Must be explicitely granted
    *If A is a friend of B
    *B is friend of C
    *then A is NOT a friend of C
